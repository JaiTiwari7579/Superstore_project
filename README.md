# 📊 Superstore Sales Data Analytics Project

## 🚀 Overview
This is an end-to-end data analytics project where raw data is transformed into meaningful business insights using Python, SQL, and Power BI.

---

## 🔍 Project Workflow
- Data Cleaning & Analysis using Python (Pandas)
- Customer Segmentation using RFM Analysis
- Sales Forecasting using Moving Average
- Business Insights using SQL (PostgreSQL)
- Interactive Dashboard creation using Power BI

---

## 🛠 Tools & Technologies
- Python (Pandas, Matplotlib)
- SQL (PostgreSQL)
- Power BI
- Excel / CSV Dataset

---

## 📊 Key Insights
- Consumer segment generates the highest revenue
- West region contributes the most sales
- Technology category is the top performer
- Higher discounts lead to lower profit margins
- Identified loss-making products and segments

---

## 📂 Files Included
- `superstore_data.csv` → Dataset  
- `analysis.ipynb` → Python analysis (RFM + Forecasting)  
- `queries.sql` → SQL queries for insights  
- `dashboard.pbix` → Power BI dashboard  
- `dashboard.png` → Dashboard preview
